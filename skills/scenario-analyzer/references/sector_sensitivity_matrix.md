# Sector Sensitivity Matrix

This reference organizes sector-specific sensitivities to various event types in a matrix format. Use this when performing scenario analysis to quickly identify which sectors are most likely to be affected.

## Legend

**Impact Level:**
- `++` : Strong positive impact
- `+` : Positive impact
- `0` : Neutral / minimal impact
- `-` : Negative impact
- `--` : Strong negative impact

**Confidence Level:**
- `H` : High (consistent historical pattern)
- `M` : Medium (situation-dependent)
- `L` : Low (high uncertainty)

## How to Use

1. **Identify the Event Type**: Determine the event category from the news headline.
2. **Consult the Corresponding Matrix**: Locate the relevant matrix from the list above.
3. **Analyze Impact and Confidence**: Assess the potential sector-specific directions and historical certainty levels.
4. **Use Sample Stocks as a Starting Point**: Initiate deeper fundamental/technical analysis on the listed companies.
5. **Combine Matrices for Complex Scenarios**: If multiple events occur, integrate findings from multiple matrices.

## Important Notes

- **Individual Company Variation**: Company-specific fundamentals can override sector-level trends.
- **Timing Differentiation**: Distinguish between immediate reactions and delayed structural impacts.
- **Scale Matters**: The overall severity of the impact depends on the scale of the event.
- **Market Pricing**: Revisions are minimal if the event has already been fully priced in by the market.
